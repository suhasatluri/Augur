# Augur

> Read the signal before it speaks.

Swarm intelligence platform for financial market simulation.

**V1: ASX Earnings Surprise Predictor** — spawns 50 autonomous 
analyst agents, seeds them with public ASX data, runs a 
multi-round debate, returns a probability distribution of 
earnings beat/miss before the company reports.

## Status
🔨 Active development — V1 in progress

## Stack
- Python FastAPI · Neon PostgreSQL · Claude API
- Next.js 14 · Cloudflare R2 · Railway
